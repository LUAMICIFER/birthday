## Mandotry Vars 

   - `API_ID` :  Give API_ID of your Alternate Telegram Account.
   - `API_HASH` :  Give API_HASH of your Alternate Telegram Account.
   - `STRING SESSION` :  Make a string session from [here](https://replit.com/@sahilsi552/QNR-3#main.py)
